## ğŸ—ï¸ Architecture

MLS follows an **Atomic Design** pattern:

### Design System
- **Atoms** - Basic building blocks (Button, Badge, Input)
- **Molecules** - Simple combinations (SearchInput, ConfirmDialog)
- **Organisms** - Complex UI sections (TaskTable, ApiKeyTable)
- **Templates** - Page layouts (AdminShell)

### Features
Complete, self-contained feature modules:
- Backup management
- Task monitoring
- Settings configuration
- Search testing

### Golden Paths
Zero-config components for instant productivity:
- QuickStart
- AdminConsole
- SearchPlayground
